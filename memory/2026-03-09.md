## 📱 투자커멘드 UI/UX 최적화 기획

### 이란 대시보드 적용 기능
| 기능 | 적용 대상 |
|------|----------|
| 이동 배너 | 긴급 뉴스/속보 티커 |
| 위험도 알림 | 포트폴리오 리스크 경고 |
| 그리드 카드 | 총자산/ISA/IRP/연금 |
| 진행바 | 목표 수익률 달성률 |
| 수평 스크롤 | 보유 종목 카드 뷰 |
| 스파크라인 | 미니 주가 차트 |
| 접기/펼치기 | 상세 정보 섹션 |
| 색상 코딩 | 수익(빨강)/손실(초록) |

### 화멵별 개선 계획
1. **홈**: 포트폴리오 그리드 + 티커 + 목표 게이지
2. **WiseReport**: 리포트 카드 + 필터 탭 + 카운트다운
3. **포트폴리오**: 도넛차트 + 트렌드 + 종목별 카드
4. **플레이북**: 플레이북 카드 + 실행 상태 + 진행 게이지
